为 Magic Storage 模组添加拼音搜索支持!

功能特性:
✓ 支持拼音全拼搜索 (例如: tukuai → 土块)
✓ 支持拼音首字母搜索 (例如: tk → 土块)
✓ 完全兼容原有的中文/英文搜索
✓ 无缝集成,零配置即用

使用方法:
在 Magic Storage 的搜索框中直接输入拼音即可!

注意: 本 Mod 需要 Magic Storage 才能运行。

本mod由ai辅助完成，之所以创建本mod，是因为原有mod以及很久没有更新，似乎存在一些问题
代码已经开源，核心代码不足200行，如果有需要可以自行进行修改完善：
github : https://github.com/Sisyphean-a/MagicStoragePinyinHelper

理论上，如果魔法存储不变更搜索逻辑，此mod就不需要更新
